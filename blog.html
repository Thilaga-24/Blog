    HTML

  1. HTML describes the structure of a web page semantically and originally included cues for its appearance. HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects such as interactive forms may be embedded into the rendered page.
  2. HTML (HyperText Markup Language) is the code that is used to structure a web page and its content. For example, content could be structured within a set of paragraphs, a list of bulleted points, or using images and data tables.
  3. The declaration defines that this document is an HTML5 document
  4. The element is the root element of an HTML page
  5. The element contains meta information about the HTML page
  6. The element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

    CSS

  1. CSS is the acronym of “Cascading Style Sheets”. CSS is a computer language for laying out and structuring web pages (HTML or XML). This language contains coding elements and is composed of these “cascading style sheets” which are equally called CSS files.
  2. CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.
  3. The selector points to the HTML element you want to style.
  4. The declaration block contains one or more declarations separated by semicolons.
  5. Each declaration includes a CSS property name and a value, separated by a colon.
  6. Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.
